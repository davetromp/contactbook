Readme

This is a proto type web application that displays a contact list.

The data consists of:
  • Firstname
  • Lastname
  • Street
  • Zip
  • City
  • Image

The data set is hosted in Google Docs as a spreadsheet: https://docs.google.com/spreadsheets/d/1HclH8WugumpyWvNHfE1zopZGeA38DdCdM3458VjBeNg/edit?usp=sharing

The spreadsheet can be edited in the spreadsheet and then it can be imported in the application by clicking the Import link in the menu bar. The contacts can be accessed via an API. You can inspect the API by clicking the Api link in the menu bar. Any links to images that are valid imagelinks are transformed into thumbnails in the form of base64 data strings and are made available as such via the Api.
New contacts can be posted via the Api once logged in.
The log in credentials are:
user: demo
pw: contactbook

url: /api/contacts/

The contacts information is searchable via the Api like so:
/api/search/?q=Dave
